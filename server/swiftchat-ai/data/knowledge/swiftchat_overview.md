# SwiftChat Overview

## What SwiftChat is
SwiftChat is the chat-first education super app used by Gujarat schools, Cluster Resource Centres, District Education Offices, the State Secretariat, and PFMS payment officers. Every action — marking attendance, scanning XAMTA worksheets, opening dashboards, filing scholarship applications, retrying payments — runs as a conversation in the chat thread, with complex forms and tables opening in a right-side canvas (full-screen sheet on mobile).

## Supported roles
- **Teacher** — marks attendance, scans XAMTA, files DigiVritti scholarship applications, fixes rejected forms.
- **Principal** — school-wide dashboards, read-only scholarship summary.
- **CRC (Cluster Resource Centre)** — also called Cluster Approver. Reviews, approves or rejects scholarship applications submitted by teachers in the cluster.
- **DEO (District Education Officer)** — district-wide analytics, payment failure tracking, attendance review, escalation.
- **State Secretary** — state-wide funnel, district comparison, bulk approval, policy what-if scenarios.
- **PFMS Officer** — payment queue, retry failed payments, batch status.
- **Parent** — sees only their own child's attendance, results, homework, and can message the teacher.

## Modules
- Mark Attendance
- XAMTA Scan (assessment scoring)
- Class Dashboard / School / District / State dashboards
- DigiVritti (scholarship hub: Namo Lakshmi + Namo Saraswati)
- Reports & analytics
- Parent alerts (WhatsApp/SMS)

## Navigation
The chat input is the primary command surface. Quick-action tiles appear on the home page per role. Typing natural language — English, Hindi, Hinglish, Gujarati — also works because of the global SwiftChat NLP layer; the system understands phrases like "attendance kholo", "Class 7 ni hajri lo", or "rejected students dikhao" and routes the user to the right module.
