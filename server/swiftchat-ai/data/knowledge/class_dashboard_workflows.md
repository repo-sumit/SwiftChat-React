# Class / School / District / State Dashboards

## Class Dashboard
Available to Teacher, Principal, and CRC. Shows for the selected class:
- Today's attendance %
- Subject-wise averages (Math, Science, Gujarati)
- Top 5 / bottom 5 students
- Learning-outcome coverage from the latest XAMTA scan
- At-risk student list (chronic absence + low scores)

Open with "Class 6 dashboard", "class report dikhao", "performance dikhao", or by tapping the Class Dashboard tile.

## School Dashboard
Principal and CRC can open the School Dashboard. It aggregates all classes:
- School attendance %
- Class-by-class comparison
- Subject mastery heat-map
- Pending DigiVritti applications (read-only summary)
- Teacher coverage (which teachers have marked attendance today)

## District Dashboard
DEO and State Secretary. Aggregates all schools in the district:
- Approval rate, payment success rate, beneficiary count
- Top 5 worst-performing blocks
- Failed-payment causes
- Pending CRC backlog by cluster
- Aadhaar correction queue

## State Dashboard
State Secretary only. Aggregates all 33 districts:
- Total beneficiaries by scheme
- Application funnel (initiated → submitted → auto-rejected → approved → paid)
- Sanctioned vs disbursed vs pending amount
- District ranking by payment success rate
- Bulk approval queues
- Monsoon what-if simulator (attendance threshold scenarios)

## Drill-down
Every chart on every dashboard opens a deeper view. The right-side canvas supports student lists, filterable tables, and CSV mock-export.
