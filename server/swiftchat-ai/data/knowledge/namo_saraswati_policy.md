# Namo Saraswati Vigyan Sadhana — Policy

## Scheme summary
₹25,000 disbursement over two academic years (Class 11 + Class 12) for girl students opting for the **Science** stream in government-recognised schools across Gujarat. Goal: encourage girls to take Science and continue into STEM higher education. Funds go via DBT to the mother's bank account.

## Eligibility
- Student is **female**.
- Currently enrolled in **Class 11 or Class 12** with **Science** as the chosen stream.
- **Class 10 board result ≥ 50%** in the qualifying examination. The Exam Board API verifies seat number → result automatically; below 50% auto-rejects.
- Mother / guardian has a valid Aadhaar + bank account, and the **mother's name on Aadhaar must match the bank passbook**.
- A student may be dual-eligible for Namo Lakshmi and Namo Saraswati at the same time.

## Required documents
- Student Aadhaar (front and back)
- Mother's Aadhaar
- Mother's bank passbook / cancelled cheque
- Class 10 marksheet (used until the Exam Board API verification completes)
- Class 11 / 12 admission proof (school confirms via SSO)
- Student photograph

## Auto-approval rule
If Class 10 percentage ≥ 50% AND seat number is verified by the Exam Board API, the application is auto-approved by the bot at submission. Manual CRC review is reserved for borderline / unverifiable cases.

## Disbursement
The same monthly attendance-based cycle as Namo Lakshmi (≥ 80% threshold; first month exempt). Disbursement amount per month varies with the academic schedule.

## Why Class 10 ≥ 50% is required
The scheme is targeted at students likely to complete Class 12 Science successfully. The threshold filters out late-stage dropouts who would otherwise consume budget without finishing the stream.

## Common rejection reasons
- **Below 50% in Class 10** — non-correctable; the scheme is unavailable.
- **Seat number invalid** — wrong digit; teacher can correct and resubmit.
- **Stream mismatch** — student is not actually in Science.
- **Mother name mismatch** — same as Namo Lakshmi; correct the spelling on Aadhaar or bank.
